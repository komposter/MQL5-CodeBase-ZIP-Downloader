MQL5 CodeBase ZIP Downloader

Описание
- Загружает все файлы из любой страницы `mql5.com/*/code/*` в один ZIP-архив.
- Полностью сохраняет структуру папок, как указано в группах публикации (например, `MQL5/Indicators/...`).
- Игнорирует встроенную кнопку «Скачать ZIP» и собирает актуальные версии файлов по прямым ссылкам.

Установка
- Откройте `chrome://extensions`.
- Включите режим разработчика (`Developer mode`).
- Нажмите `Load unpacked` и выберите корень проекта (папку с `manifest.json`).
- Для теста локальной страницы включите «Allow access to file URLs» в настройках расширения.

Использование
- Откройте страницу CodeBase, например `https://www.mql5.com/en/code/22166` или `https://www.mql5.com/ru/code/65527`.
- Нажмите на иконку расширения или на добавленную кнопку «Скачать CodeBase ZIP» рядом с блоком вложений.
- Расширение автоматически раскрывает скрытые элементы и скачивает ZIP со строго корректными путями.

Тестовая страница
- Откройте файл `page_example/Free download of the 'TradeTransactions'...html` в Chrome.
- Нажмите на иконку расширения или на добавленную кнопку.
- Начнётся загрузка ZIP-архива с 11 файлами в 4 папках.

Примечания
- Для доступа к скачиванию используется авторизация страницы; убедитесь, что вы залогинены, если это требуется.
- Пути берутся из заголовков групп, отображаемых на странице.
